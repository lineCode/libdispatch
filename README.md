libdispatch
===========
Grand Central Dispatch for Windows, extracted from iTunes 12.0.1

This library is 32-bit though it comes from iTunes x64.

Usage
-----
1. Install MinGW
2. clang/gcc main.c -omain -Iinclude -L. -ldispatch

Dependency
----------
Microsoft Visual C++ 2010 x86
