libdispatch
===========
Grand Central Dispatch for Windows, extracted from iTunes 11.1.5

This library is 32-bit though I extracted it from iTunes x64.

Usage
-----
1. Install MinGW
2. clang/gcc main.c -omain -Iinclude -L. -ldispatch

Dependency
----------
Microsoft Visual C++ 2010 x86
